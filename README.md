# Fintech Review Analytics

## Project Overview
A rigorous analytics pipeline that transforms raw Google Play Store reviews for three Ethiopian banks into actionable insights for product teams.

**Banks Analyzed:** Commercial Bank of Ethiopia (CBE), Bank of Abyssinia (BOA), Dashen Bank

## Tasks
| Task | Description | Branch |
|------|-------------|--------|
| Task 1 | Data collection & preprocessing | task-1 |
| Task 2 | Sentiment & thematic analysis | task-2 |
| Task 3 | PostgreSQL database engineering | task-3 |
| Task 4 | Insights & recommendations | task-4 |

## How to Reproduce

1. Clone the repository:
   git clone https://github.com/YOUR_USERNAME/fintech-review-analytics.git

2. Create and activate virtual environment:
   python -m venv venv
   source venv/Scripts/activate

3. Install dependencies:
   pip install -r requirements.txt

## Data
Raw data is excluded from version control (.gitignore). Reviews are scraped from Google Play Store using google-play-scraper.
